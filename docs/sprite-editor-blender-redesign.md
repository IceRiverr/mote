# 精灵编辑器 Blender 风格重设计

## 设计原则

按照 Blender 的设计哲学重构精灵编辑器：
1. **模式化编辑** - 通过模式切换改变整个编辑器上下文
2. **可视化优先** - 所有编辑操作都在画布中直接可见
3. **属性面板** - 右侧 N-Panel 显示选中对象的详细属性
4. **工具栏** - 左侧 T-Panel 提供当前模式的上下文工具
5. **快捷键驱动** - 支持快捷键快速切换模式和工具

## 三种编辑模式

### 1. 选择模式 (Select Mode) — 默认
- **快捷键**: `1`
- **功能**: 选择/多选帧
- **画布行为**: 
  - 左键单击: 选择帧
  - Ctrl+左键: 多选/取消选择
  - Shift+左键: 范围选择
  - 中键/Alt+左键: 平移画布

### 2. 碰撞编辑模式 (Collider Mode)
- **快捷键**: `2`
- **功能**: 编辑帧的碰撞体
- **画布行为**:
  - 显示所有碰撞体覆盖层（红色半透明）
  - 左键点击帧: 应用当前工具
  - 十字准星光标
- **工具栏**:
  - 预设: 完整、上半、下半、斜坡(NE/NW/SE/SW)
  - 擦除工具
  - "应用到选中" 按钮

### 3. 标签模式 (Tag Mode)
- **快捷键**: `3`
- **功能**: 批量编辑标签
- **画布行为**: 选择帧并在属性面板编辑标签

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 循环切换模式 (选择→碰撞→标签→选择) |
| `1` | 切换到选择模式 |
| `2` | 切换到碰撞编辑模式 |
| `3` | 切换到标签编辑模式 |
| `N` | 显示/隐藏属性面板 |
| `Ctrl+滚轮` | 缩放 |
| `中键拖拽` | 平移画布 |

## 属性面板 (N-Panel)

位置: 右侧，可开关

### 帧信息
- 名称
- 尺寸 (宽×高)
- 位置 (在图集中的坐标)

### 碰撞体
- 预设单选按钮组
- 当前碰撞形状数量显示

### 标签
- 标签列表 (可删除)
- 添加标签输入框

---

## 状态栏

位置: 底部

显示内容:
- 当前模式 (带颜色标识)
- 选中帧信息
- 操作提示
- 状态消息

## 与原设计的对比

| 功能 | 旧设计 (右键菜单) | 新设计 (Blender风格) |
|------|------------------|---------------------|
| 添加碰撞体 | 右键→选择预设 | 选中帧→点击"碰撞模式"→点击工具/帧 |
| 批量添加 | 不可行 | 多选帧→点击预设→应用到所有 |
| 查看碰撞体 | 小方块标记 | 始终显示半透明覆盖层 |
| 编辑标签 | 右键菜单输入 | 属性面板可视化编辑 |
| 模式切换 | 无 | Tab/数字键快速切换 |

- `SpriteEditorHeader.tsx` - 模式切换按钮
- `SpriteEditorCanvas.tsx` - 模式感知的交互和渲染
- `FrameContextMenu.tsx` - 简化为导航菜单
